### What I learned

While the HTML structure was straightforward, this project provided a great opportunity to practice fine-tuning the internal layout of a card component. I spent time ensuring the spacing and alignment of the text elements were consistent with the design.

The biggest learning curve for this project was **deployment**. As this was my first time deploying a project to the web, I learned:
- How to manage file paths for hosting environments (Relative vs. Absolute paths).
- The process of pushing code to GitHub and using GitHub Pages for hosting.
- Debugging 404 errors related to case-sensitive file naming and directory structures.

### Continued development

In my upcoming projects, I plan to focus on:
- **Advanced CSS Layouts:** Mastering Grid and complex Flexbox patterns.
- **Animations:** Adding subtle transitions to improve UX.
- **Responsive Design:** Ensuring components look perfect on every screen size.
- **Dynamic Content:** Moving beyond static sites to work with APIs and AJAX.

### AI Collaboration

I used Gemini as an AI collaborator during this project. This was particularly helpful for:
- **Debugging Deployment:** Identifying why images weren't loading on the live site.
- **Best Practices:** Understanding the importance of `min-height: 100vh` for vertical centering.
- **Documentation:** Refining this README to be more professional.
